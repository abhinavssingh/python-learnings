# Module 2 – Maths

This module focuses on **mathematical foundations required for data science**, including **linear algebra and probability**, implemented using Python, NumPy, and SciPy.

Each script goes beyond simple computation by generating **step‑by‑step explanations** and **interactive HTML reports**, making abstract mathematical concepts easier to understand and visualize.

---

## 📌 Objective of This Module

The goals of this Maths module are to:

- Strengthen understanding of **linear algebra concepts**
- Apply **probability theory** to real datasets
- Bridge mathematical theory with **practical Python implementation**
- Visualize mathematical results using charts and LaTeX
- Build intuition required for **data science, machine learning, and analytics**

---

## 📂 Topics Covered

The scripts in this folder are organized around the following core areas:

---

### 🧮 1. Linear Algebra (Matrix Operations)

**Concepts covered:**
- Matrix dot product
- Vector cross product
- Determinant
- Adjoint (Adjugate)
- Inverse of a matrix
- Row Echelon Form (RREF)
- Rank of a matrix
- Eigenvalues and Eigenvectors

**Key highlights:**
- Step‑by‑step derivations using LaTeX
- Comparison between:
  - Custom mathematical logic
  - NumPy’s built‑in functions (`np.dot`, `np.linalg.inv`, etc.)
- Clean visual presentation of each operation

These concepts are foundational for:
- Machine learning algorithms
- Optimization
- Computer vision
- Scientific computing

---

### 📊 2. Probability & Statistics

**Distributions demonstrated:**
- Bernoulli
- Binomial
- Poisson
- Normal (Gaussian)
- Uniform

**Using:**
- Real retail store data
- Empirical probability estimation
- Kernel Density Estimation (KDE)
- Theoretical PMFs and PDFs

**Key highlights:**
- Business‑oriented probability use cases
- Side‑by‑side comparison of:
  - Theory
  - Actual observed data
- Interactive Plotly charts for better intuition

These topics are essential for:
- Predictive modeling
- Risk analysis
- Business analytics
- Data science interviews

---

All scripts in this module generate **self‑contained HTML reports**, which include:
