# NumPy Basics – Array Reshape & Transpose Report

This script demonstrates **fundamental NumPy array operations** such as creation, reshaping, and transposing arrays of different data types, and generates a **simple HTML report** to visualize and verify the transformations.

It is intended as an **introductory learning exercise** for understanding NumPy’s core array mechanics.

---

## 📌 Purpose of This Script

The primary objectives of this script are to:

- Illustrate basic NumPy array creation
- Demonstrate `reshape()` and `transpose()` operations
- Compare behavior across different data types:
  - Integers
  - Floating‑point numbers
  - Strings
- Render results in a structured, readable **HTML report**
