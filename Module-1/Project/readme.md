# Project: Customer Data Analysis using Flask Application

## Overview
This project combines data analysis with a Flask web application for managing customer information and orders.

## Project Structure & Objectives

### 1. **requirements.md**
- **Objective**: Document all project dependencies and system requirements
- **Execute**: First - Review and install all required packages

### 2. **customer_model.ipynb** 
- **Objective**: Jupyter notebook for setting up the Flast app and Databse model
- **Execute**: Second - Generates app.py and Model.py

### 3. **models.py**
- **Objective**: Define database models and data structures for customers orders. 

### 4. **app.py**
- **Objective**: Flask application entry point; handles server routing and request management

### 5. **customer_order_form** 
- **Objective**: This notebool launch a Form in a browser uisng Flask application to create customer dataset.
- **Execute**: Third - Creating Dataset.

### 6. **templates/form.html**
- **Objective**: HTML template for the customer order form interface
- **Execute**: Used by app.py when serving the form page

### 7. **customer_data_insights.ipynb**
- **Objective**: Analyze customer data patterns and generate insights
- **Execute**: Fourth - Data exploration and analysis

### 8. **customer_report.html**
- **Objective**: Generate and display comprehensive HTML reports

## Execution Sequence
1. Install dependencies (requirements.md)
2. Configure models (models.py)
3. Launch application (app.py)
4. Test functionality (notebooks)
5. Generate reports (customer_report.html)
