# Project Requirements

- Build a text-based adventure game in Python where the player explores locations, makes choices, and completes a simple quest. 
- Tooling: Use VS Code with GitHub Copilot extension (if available). 
- Dataset: None required. 
- Provide a CLI game that: asks for the player’s name, offers an initial choice between a dark forest and a mysterious cave, and branches into further decision points. 
- Include three outcomes:
    - Win – find the treasure,
    - Lose – poor decision ends the adventure,
    - Restart – player can replay after an unsuccessful attempt. 
- Implement the following tasks:
    - Task 1: Project setup with a file named adventure_game.py. 
    - Task 2: start_game() — intro + name + initial forest/cave choice. 
    - Task 3: forest_path() — choices: follow a river or climb a tree. 
    - Task 4: cave_path() — choices: light a torch or proceed in the dark. 
    - Task 5: Game loop that allows replay until completion. 
- Deliverables: the script and a short report describing Copilot assistance, challenges, and enhancements.
